# An_Azure_app_with_VMSS-AGW
This is all about deploying an Azure app in azure virtual machines using Virtual Machine Scale Sets (VMSS) and Application Gateway to promote scalability and high availability.

## Internet <=> Azure Application gateway (Layer 7) <=> Network Security Group (NSG) <=> VMSS <=> Instance

### Azure components
1. Application Gateway
2. VMSS (Virtual Machine Scale Sets)
3. Vnet, Subnet, Azure Bastion, NAT Gateway, Public IP
4. Network Security Group
5. Azure Subscription and Resource group
